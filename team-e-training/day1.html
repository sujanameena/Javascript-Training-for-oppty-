CSS Overflow Property Examples

1. overflow: visible

The default value. Content is not clipped and renders outside the box.

This content is much larger than the box height of 100px. Notice how it "spills" out of the red borders and continues downward, potentially overlapping other elements.
overflow: visible;

2. overflow: hidden

The content is clipped at the border and the rest is invisible.

This content is much larger than the box height of 100px. However, because we set overflow to hidden, you cannot see the rest of this text at all. It is cut off.
overflow: hidden;

3. overflow: scroll

Adds scrollbars to the box, even if the content fits inside.

This box adds scrollbars. Users can scroll to see the hidden content. Note that both vertical and horizontal scrollbars often appear depending on the browser.
overflow: scroll;

4. overflow: auto

Adds scrollbars only if the content exceeds the box size.

This is the most common choice. Scrollbars only appear because this text is too long for the 100px height. If the text was short, no scrollbars would show.
overflow: auto;
