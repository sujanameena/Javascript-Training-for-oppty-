Objects in JavaScript: --------------------- In JavaScript, an object is a collection of related data and functionality, stored as key–value pairs. Keys are called properties (for data) or methods (for functions). There are two primary ways to create an object in JavaScript: 1. Creation Using Object Literal: Example:1 2. Accessing Object Properties: Accessing object properties means reading the values stored in an object using its keys. You can use **dot notation** (`obj.key`) or **bracket notation** (`obj["key"]`). Example:1 2. Modifying Object Properties: Modifying object properties means changing the values of existing keys in a JavaScript object. You can update properties using dot notation (obj.key = value) or bracket notation (obj["key"] = value). Example:1 Example:2 3. Adding Properties to an Object: Adding properties to an object means creating new key–value pairs in an existing JavaScript object. You can add properties using dot notation (obj.newKey = value) or bracket notation (obj["newKey"] = value). Example:1 Example:2 4. Removing Properties from an Object: Removing properties from an object means deleting a key–value pair from an existing JavaScript object. You can remove properties using the delete operator (delete obj.key or delete obj["key"]). Example:1 Example:2 5. Checking if a Property Exists: To check if a property exists in a JavaScript object, you can use either the in operator or the hasOwnProperty() method. Example:1 Example:2 6. Iterating Through Object Properties: Iterating through object properties means accessing each key–value pair in a JavaScript object, usually to perform actions like displaying, modifying, or analyzing the data. Example:1 Example:2 7. Merging Objects: Merging objects means combining the properties of two or more JavaScript objects into a single object. This is often done using the spread operator (...) or Object.assign(). Example:1 Example:2 8. Object Length: Object length refers to the number of properties (keys) present in a JavaScript object. Since objects do not have a built-in length property, you can use Object.keys(obj).length to get the count.
