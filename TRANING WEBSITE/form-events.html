Form Events

Form events allow you to control how users interact with input fields, select boxes, and submission buttons. They are the primary way to validate data before sending it to a server.

1. The submit Event (and preventDefault)

The submit event occurs when a user clicks a submit button or presses Enter inside a form. Crucially, we use event.preventDefault() to stop the browser from refreshing the page, allowing JavaScript to handle the data.

<!DOCTYPE html> <html> <body> <form id="loginForm"> <input type="text" placeholder="Username" required> <button type="submit">Login</button> </form> <p id="status"></p> <script> document.getElementById("loginForm").addEventListener("submit", (e) => { e.preventDefault(); // Stop page refresh document.getElementById("status").innerText = "Form intercepted by JS!"; }); </script> </body> </html>

2. Real-time Feedback (input Event)

The input event triggers every single time the value of an <input> or <textarea> changes. This is perfect for character counters or live search bars.

<!DOCTYPE html> <html> <body> <input type="text" id="user" placeholder="Type something..."> <p>Character Count: <span id="count">0</span></p> <script> document.getElementById("user").addEventListener("input", (e) => { document.getElementById("count").innerText = e.target.value.length; }); </script> </body> </html>

3. The change Event

Unlike input, the change event only fires when the element loses focus (blur) after the value was modified, or when a selection is made in a <select> menu.

<!DOCTYPE html> <html> <body> <select id="colorPicker"> <option value="white">White</option> <option value="lightblue">Blue</option> <option value="lightgreen">Green</option> </select> <script> document.getElementById("colorPicker").addEventListener("change", (e) => { document.body.style.backgroundColor = e.target.value; }); </script> </body> </html>

4. Focus and Blur (Styling Inputs)

<!DOCTYPE html> <html> <body> <input type="email" id="email" placeholder="Email Address"> <p id="hint" style="display:none;">We will never share your email!</p> <script> const email = document.getElementById("email"); const hint = document.getElementById("hint"); email.addEventListener("focus", () => { hint.style.display = "block"; email.style.border = "2px solid blue"; }); email.addEventListener("blur", () => { hint.style.display = "none"; email.style.border = "1px solid gray"; }); </script> </body> </html>

5. The reset Event

<!DOCTYPE html> <html> <body> <form id="survey"> <input type="text" placeholder="Your Feedback"> <button type="reset">Reset Form</button> </form> <script> document.getElementById("survey").addEventListener("reset", () => { alert("All fields are being cleared!"); }); </script> </body> </html>
