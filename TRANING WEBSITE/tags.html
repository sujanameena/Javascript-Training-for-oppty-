Common HTML Tags

HTML tags are the building blocks of web pages. They tell the browser how to display content.

1. Root & Structural Tags

These tags form the essential skeleton of every HTML document. They define the document's boundaries and basic layout.

<!DOCTYPE html> <html> <head> <title>Page Title</title> </head> <body> <h1>Hello World</h1> </body> </html>
  • <html>: The root element that wraps all other tags on the page.
  • <head>: A container for metadata (data about data), such as the title and links to CSS.
  • <body>: Contains all visible content like text and images.
  • <!DOCTYPE html>: Informs the browser that the document is HTML5.

2. Metadata & Scripting Tags

These tags provide "behind the scenes" info or add interactivity. They usually live inside the <head>.

<head> <title>My Website</title> <meta name="description" content="Tutorial page"> <link rel="stylesheet" href="styles.css"> <script src="script.js"></script> </head>
  • <title>: Sets the name of the page in the browser tab.
  • <meta>: Provides keywords, descriptions, and author info.
  • <link>: Connects external resources like CSS stylesheets.
  • <script>: Links JavaScript to make the page interactive.

3. Text Formatting Tags

Use these to control text appearance and give specific meanings (Logical vs. Physical).

<p>This is <strong>important</strong> text.</p> <p>This is <em>emphasized</em> text.</p> <span style="color:red">Specific styled text</span>
  • <strong> and <b>: Bold text. Strong implies importance; B is just for style.
  • <em> and <i>: Italics. Em adds emphasis; I is just for style.
  • <span>: An inline container used to style a specific part of text.

4. Layout & Semantic Tags (Sectioning)

Semantic tags describe the content's meaning for search engines and screen readers.

<header>Header Content</header> <nav>Menu Links</nav> <section>Main Chapter</section> <footer>Copyright 2026</footer>
  • <div>: A generic block-level container for grouping.
  • <header>: Introductory content at the top of a page or section.
  • <nav>: Specifically for navigation blocks (menus).
  • <footer>: The bottom of a page, containing contact or copyright info.

5. Media & Interactive Tags

These tags embed multimedia or collect data from users.

<a href="https://google.com">Visit Google</a> <img src="logo.png" alt="Company Logo"> <video controls> <source src="movie.mp4" type="video/mp4"> </video>
  • <a> (Anchor): Creates hyperlinks to other pages.
  • <img>: Embeds images (this is an "empty" tag).
  • <form>: A container for input elements to collect user data.
  • <video> and <audio>: Play media files directly in the browser.
