JSON (parse & stringify)

JSON (JavaScript Object Notation) is the standard format for exchanging data between a browser and a server. Since a server can only send data as text, JavaScript provides two essential methods to convert between "Text" and "Objects".

The Rule of JSON: JSON is a string format. To use it in JavaScript logic, you must Parse it. To send it to a server, you must Stringify it.

1. JSON.parse() vs. JSON.stringify()

These two methods are opposites. One "unpacks" data, while the other "packs" it for travel.

Method Direction Result Use Case
JSON.parse() String ➔ Object JavaScript Object Receiving data from an API.
JSON.stringify() Object ➔ String JSON String Sending data to a server.

2. Important Syntax Rules

JSON looks like a JavaScript object, but it is much stricter:

  • Double Quotes: All keys and string values must use double quotes ("name"), not single quotes.
  • No Functions: JSON cannot store functions or methods.
  • Data Types: Allowed types are String, Number, Object, Array, Boolean, and Null.
  • Trailing Commas: Unlike JS arrays, the last item in a JSON object cannot have a comma.

3. Runnable Examples

Example 1: Converting Text to an Object (Parse)

When you get data from a server, it arrives as a string. We parse it to access its properties.

<!DOCTYPE html> <html> <body> <p id="demo"></p> <script> const text = '{"name":"John", "age":30, "city":"New York"}'; const obj = JSON.parse(text); document.getElementById("demo").innerHTML = "Name: " + obj.name + ", Age: " + obj.age; </script> </body> </html>

Example 2: Converting an Object to Text (Stringify)

Before sending data to an API, you must turn your JS object into a JSON string.

<!DOCTYPE html> <html> <body> <p id="demo2"></p> <script> const user = { name: "Alice", isAdmin: true, score: 95 }; const myJSON = JSON.stringify(user); document.getElementById("demo2").innerText = myJSON; console.log(typeof myJSON); // Returns "string" </script> </body> </html>

Example 3: Working with Arrays in JSON

JSON can also handle lists of items (arrays).

<script> const jsonArr = '["HTML", "CSS", "JS", "React"]'; const myArr = JSON.parse(jsonArr); document.write("First Item: " + myArr[0]); // Accessing like a normal array after parsing! </script>

Example 4: Stringify with Formatting (Pretty Print)

You can add space to `stringify` to make the text readable for humans in logs.

<!DOCTYPE html> <html> <body> <pre id="out"></pre> <script> const data = { id: 1, task: "Learn JSON", status: "Done" }; // The '4' adds indentation spaces const pretty = JSON.stringify(data, null, 4); document.getElementById("out").innerText = pretty; </script> </body> </html>

Example 5: Deep Cloning an Object

A famous JS trick is using JSON to create a perfect copy of an object so that changing the copy doesn't affect the original.

<script> const original = { x: 10, y: 20 }; // Convert to string and back to object to "disconnect" them const clone = JSON.parse(JSON.stringify(original)); clone.x = 99; document.write("Original X: " + original.x); // Still 10 document.write("<br>Clone X: " + clone.x); // 99 </script>
Oppty Academy Tip: Always use try...catch when using JSON.parse(). If the string is not valid JSON (e.g., it has a missing bracket), your whole script will crash without a catch block!
