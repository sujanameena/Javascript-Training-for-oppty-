JavaScript Timing Events

The window object allows execution of code at specified time intervals. These time intervals are called timing events. The two key methods to use with JavaScript are setTimeout() and setInterval().

1. The setTimeout() Method

The setTimeout() method executes a function, after waiting a specified number of milliseconds.

<!DOCTYPE html> <html> <body> <h2>JavaScript setTimeout</h2> <p>Wait 3 seconds for the greeting:</p> <h2 id="demo"></h2> <script> const myTimeout = setTimeout(myGreeting, 3000); function myGreeting() { document.getElementById("demo").innerHTML = "Happy Coding!" } </script> </body> </html>

2. The setInterval() Method

The setInterval() method repeats a given function at every given time-interval.

<!DOCTYPE html> <html> <body> <h2>JavaScript setInterval</h2> <p>A script to display the current time every second:</p> <p id="demo"></p> <script> setInterval(myTimer, 1000); function myTimer() { const d = new Date(); document.getElementById("demo").innerHTML = d.toLocaleTimeString(); } </script> </body> </html>

3. Stopping Execution

The clearTimeout() and clearInterval() methods are used to stop the execution of the functions specified in the timer methods.

<!DOCTYPE html> <html> <body> <h2>Stopping a Timer</h2> <p id="demo">0</p> <button onclick="stopCount()">Stop Count</button> <script> let count = 0; const myInterval = setInterval(counter, 1000); function counter() { count++; document.getElementById("demo").innerHTML = count; } function stopCount() { clearInterval(myInterval); } </script> </body> </html>
