HTML Tables

HTML tables allow web developers to arrange data into rows and columns.

1. Basic HTML Table

A basic table is defined using the <table> tag. Inside, data is organized into rows using the <tr> (Table Row) tag, and individual data cells are defined using the <td> (Table Data) tag.

<table border="1"> <tr> <td>Emil</td> <td>Tobias</td> </tr> <tr> <td>Linus</td> <td>16</td> </tr> </table>
  • Structure: Every table must have at least one row and one cell.
  • Default Styling: By default, table data is left-aligned and has no visible border unless specified via CSS (or the legacy border attribute).

2. Adding Table Headers

To make a table readable, headers are used to label columns or rows.

<table border="1"> <tr> <th>Name</th> <th>Age</th> </tr> <tr> <td>Jill</td> <td>25</td> </tr> </table>
  • Tag: Use <th> (Table Header) instead of <td> for header cells.
  • Visual Difference: Browsers typically render <th> content as bold and centered by default.
  • Scope: You can use the scope attribute (col or row) to tell screen readers whether the header is for a column or a row, improving accessibility.

3. Table with Colspan

The colspan attribute stands for "column span." It is used when you want a single cell to stretch horizontally across multiple columns.

<table border="1"> <tr> <th colspan="2">Phone Numbers</th> </tr> <tr> <td>555-0123</td> <td>555-0124</td> </tr> </table>
  • Usage: <td colspan="2"> will make that cell take up the space of two columns.
  • Common Use Case: Creating a "Total" row at the bottom of a financial table or a main heading that centers across the entire table width.

4. Table with Rowspan

The rowspan attribute is the vertical counterpart to colspan. It allows a single cell to span across multiple rows.

<table border="1"> <tr> <th>Name:</th> <td>Bill Gates</td> </tr> <tr> <th rowspan="2">Phone:</th> <td>555-1234</td> </tr> <tr> <td>555-8765</td> </tr> </table>
  • Usage: <td rowspan="3"> will make the cell extend downwards across three rows.
  • Common Use Case: When the first column of a table contains a category name that applies to several rows of data.

5. Table Captions

A caption serves as a title or a brief description for the table, helping users and search engines understand what the data represents.

<table border="1"> <caption>Monthly Savings</caption> <tr> <th>Month</th> <th>Savings</th> </tr> <tr> <td>January</td> <td>$100</td> </tr> </table>
  • Tag: Defined using the <caption> tag.
  • Placement: The <caption> tag must be inserted immediately after the opening <table> tag.
  • Alignment: By default, the caption appears centered at the very top of the table.

Summary Table of Tags

Tag Description
<table>The main container for the table.
<tr>Defines a single row.
<th>Defines a header cell (bold/centered).
<td>Defines a standard data cell.
<caption>Provides a title for the table.
