HTML

The language for building web pages

HTML Example:

<!DOCTYPE html> <html> <head> <title>HTML Tutorial</title> </head> <body> <h1>This is a heading</h1> <p>This is a paragraph.</p> </body> </html>

CSS

The language for styling web pages

CSS Example:

<!DOCTYPE html> <html> <head> <style> body { background-color: lightblue; } h1 { color: white; text-align: center; } p { font-family: verdana; font-size: 20px; } </style> </head> <body> <h1>My First CSS Example</h1> <p>This is a paragraph.</p> </body> </html>

JavaScript

The language for programming web pages

JavaScript Example:

<!DOCTYPE html> <html> <body> <h1>JavaScript</h1> <h2>The onclick Event</h2> <p id="demo">Click to change the layout</p> <button onclick="myFunction()">Click Me!</button> <script> function myFunction() { let x = document.getElementById("demo"); x.style.fontSize = "25px"; x.style.color = "red"; } </script> </body> </html>
