Custom Events

Custom events allow you to create your own event names and trigger them when specific application logic is met. This is highly useful for communication between different components.

1. The basic CustomEvent Constructor

To create a custom event, use the new CustomEvent() constructor. You give it a name and then "fire" it using the dispatchEvent() method.

<!DOCTYPE html> <html> <body> <h2>Simple Custom Event</h2> <button id="trigger">Fire 'hello' Event</button> <script> const btn = document.getElementById("trigger"); btn.addEventListener("hello", () => { alert("The 'hello' event was triggered!"); }); btn.addEventListener("click", () => { const myEvent = new CustomEvent("hello"); btn.dispatchEvent(myEvent); }); </script> </body> </html>

2. Passing Data via 'detail'

<!DOCTYPE html> <html> <body> <button id="dataBtn">Send User Data</button> <script> const btn = document.getElementById("dataBtn"); btn.addEventListener("userLogin", (e) => { alert("Welcome, " + e.detail.username); }); btn.addEventListener("click", () => { const loginEvent = new CustomEvent("userLogin", { detail: { username: "JohnDoe123" } }); btn.dispatchEvent(loginEvent); }); </script> </body> </html>

3. Bubbling Custom Events

<!DOCTYPE html> <html> <body> <div id="parent" style="padding:20px; border:1px solid red"> Parent Element <button id="child">Trigger Bubbling Event</button> </div> <script> document.getElementById("parent").addEventListener("childMsg", () => { alert("Parent caught the child's message!"); }); document.getElementById("child").addEventListener("click", () => { const msgEvent = new CustomEvent("childMsg", { bubbles: true }); document.getElementById("child").dispatchEvent(msgEvent); }); </script> </body> </html>
