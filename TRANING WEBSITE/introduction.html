Introduction to JavaScript

JavaScript is a lightweight, cross-platform, single-threaded, and interpreted scripting language. It is one of the world's most popular programming languages, used primarily to make web pages interactive.

What is JavaScript?

JavaScript allows you to implement complex features on web pages, such as timely content updates, interactive maps, animated 2D/3D graphics, and scrolling video jukeboxes. It is the third layer of the standard web technology stack, alongside HTML and CSS.

1. The "Hello World" Example

The simplest way to display data in JavaScript is by using console.log() to print a message to the browser console or alert() to show a popup.

<!DOCTYPE html> <html> <body> <h2>First JavaScript Program</h2> <button onclick="sayHello()">Click Me!</button> <script> function sayHello() { alert("Hello World from JavaScript!"); console.log("Button was clicked!"); } </script> </body> </html>

2. Dynamic HTML Content

JavaScript can change the content of an HTML element by accessing the innerHTML property through the Document Object Model (DOM).

<!DOCTYPE html> <html> <body> <h2>JavaScript DOM Interaction</h2> <p id="demo">JavaScript can change this text.</p> <button onclick="changeText()">Change Content</button> <script> function changeText() { document.getElementById("demo").innerHTML = "Hello GeeksforGeeks!"; } </script> </body> </html>

3. Working with Variables

JavaScript uses variables to store data values. It is a dynamically typed language, meaning a variable can hold different data types over its lifetime.

<!DOCTYPE html> <html> <body> <h2>JavaScript Variables</h2> <p id="result"></p> <script> let x = 10; let y = 20; let sum = x + y; document.getElementById("result").innerHTML = "The sum of x and y is: " + sum; </script> </body> </html>
Pro Tip: JavaScript was originally created in just 10 days by Brendan Eich in 1995 while he was working at Netscape.
