HTML Form Elements

Forms are used to collect user input, which is most often sent to a server for processing.

1. The <input> Element

The most versatile and frequently used form element. It changes its behavior entirely based on its type attribute.

<form> <label>Name:</label> <input type="text" name="username"> <input type="submit" value="Send"> </form>
  • Text Field (type="text"): Single-line input for names or usernames.
  • Password (type="password"): Masks characters for security.
  • Radio Buttons (type="radio"): Select only one option from a set.
  • Checkboxes (type="checkbox"): Select one or more options.

2. The <label> Element

The <label> element defines a caption for a form control.

<label for="email">Enter Email:</label> <input type="email" id="email" name="user_email">
  • Accessibility: It tells screen readers what the input field is for.
  • Usability: Clicking the label text focuses the associated input field (using the for attribute).

3. The <textarea> Element

Used for multi-line text input, such as feedback or comments.

<textarea name="message" rows="4" cols="50"> Enter your comments here... </textarea>
  • Attributes: Size is defined using rows and cols.
  • Tag Type: Unlike input, this is a paired tag (requires a closing tag).

4. The <select> and <option> Elements

Used to create a drop-down list to save space on the page.

<select name="cars"> <option value="volvo">Volvo</option> <option value="saab">Saab</option> <option value="fiat">Fiat</option> </select>
  • Value Attribute: The value is the actual data sent to the server.
  • Multi-select: Add the multiple attribute to allow selecting more than one item.

5. The <fieldset> and <legend> Elements

Used to group related data within a long form for better organization.

<fieldset> <legend>Contact Information</legend> <label>Email:</label> <input type="text"> </fieldset>

Summary of Form Attributes

Attribute Description
actionThe URL where form data is sent upon submission.
method="GET"Appends data to URL (visible, used for non-sensitive data).
method="POST"Sends data in request body (hidden, used for sensitive data).
