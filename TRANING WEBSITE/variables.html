JavaScript Variables

In JavaScript, a variable is a container for storing data values. JavaScript variables can be declared in four ways: using let, const, var, or by nothing at all.

Understanding Declaration Keywords

  • var: The oldest keyword, used from 1995 to 2015. It has function scope.
  • let: Introduced in ES6 (2015). It has block scope and is the preferred way for changeable variables.
  • const: Also ES6. Used for variables that should not be reassigned.

1. Using let for Changeable Values

<script> let score = 10; score = 20; // Reassigning value document.write("Current Score: " + score); </script>

2. Using const for Fixed Values

<script> const PI = 3.14159; // PI = 3.14; // This would cause an error document.write("Value of PI: " + PI); </script>

3. Dynamic Typing in JavaScript

JavaScript is a loosely typed language, meaning the same variable can hold different data types.

<script> let data = "Hello"; // String data = 100; // Now it's a Number document.write("Value: " + data); </script>
Pro Tip: Always use const if the value should not be changed. Use let if the value will change. Avoid var in modern projects to prevent scoping issues.
