JavaScript Conditional Statements

Conditional statements are used to perform different actions based on different conditions. In JavaScript, we use these to control the flow of the program.

Summary of Conditional Statements

Statement Description Purpose
if Executes code if a specified condition is true. Simple binary check.
else Executes code if the same condition is false. Fallback option.
else if Specifies a new condition to test if the first condition is false. Multiple alternative checks.
switch Selects one of many code blocks to be executed. Efficiently handles multiple specific values.
Ternary (? :) Shorthand for an if...else statement. Inline assignments.

1. The if Statement

Use the if statement to specify a block of JavaScript code to be executed if a condition is true.

<script> let hour = 10; if (hour < 12) { document.write("Good Morning!"); } </script>

2. The if...else Statement

Use the else statement to specify a block of code to be executed if the condition is false.

<script> let age = 15; if (age >= 18) { document.write("You are eligible to vote."); } else { document.write("You are too young to vote."); } </script>

3. The else if Statement

Use else if to specify a new condition if the first condition is false.

<script> let time = 20; if (time < 10) { document.write("Good morning"); } else if (time < 20) { document.write("Good day"); } else { document.write("Good evening"); } </script>

4. The switch Statement

The switch statement is used to perform different actions based on different conditions, often replacing multiple else if blocks for readability.

<script> let day; switch (new Date().getDay()) { case 0: day = "Sunday"; break; case 1: day = "Monday"; break; case 6: day = "Saturday"; break; default: day = "Weekday"; } document.write("Today is " + day); </script>

5. Nested if Statements

You can use an if statement inside another if statement to check more specific conditions.

<script> let num = 10; if (num > 0) { if (num % 2 == 0) { document.write("Positive even number."); } } </script>

6. Ternary Operator Logic

The conditional (ternary) operator assigns a value to a variable based on a condition.

<script> let authenticated = true; let status = authenticated ? "Welcome back!" : "Please log in."; document.write(status); </script>
Pro Tip: Use the switch statement when you have many fixed values to compare against a single variable. It makes the code much cleaner than long else if chains.
