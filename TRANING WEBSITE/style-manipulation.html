JavaScript Style Manipulation

You can directly manipulate the CSS styles of an element using the JavaScript style property. This allows you to dynamically change how elements appear on the page without modifying external stylesheets.

1. Changing a Single CSS Property

To change a specific inline style of an element, access it through the style object. For example, setting the text color of an element dynamically when a user interacts with it.

<!DOCTYPE html> <html> <body> <h3>Single Style Property</h3> <p id="demo1">This is a standard paragraph.</p> <button onclick="changeColor()">Change Color</button> <script> function changeColor() { document.getElementById("demo1").style.color = "blue"; } </script> </body> </html>

2. JavaScript camelCase Syntax

In CSS, properties with multiple words are written in kebab-case (e.g., background-color, font-size). In JavaScript, hyphens are not allowed in property names, so they must be written in camelCase (e.g., backgroundColor, fontSize).

<!DOCTYPE html> <html> <body> <h3>camelCase Formatting</h3> <p id="demo2">Notice the property names used in the script.</p> <button onclick="applyCamelCase()">Change Font and Background</button> <script> function applyCamelCase() { let el = document.getElementById("demo2"); el.style.backgroundColor = "lightgray"; // replaces background-color el.style.fontSize = "20px"; // replaces font-size } </script> </body> </html>

3. Changing Multiple Styles at Once

If you need to apply several styles simultaneously, you can set them line by line targeting the same element reference.

<!DOCTYPE html> <html> <body> <h3>Applying Multiple Styles</h3> <div id="demo3">I will become a stylized box.</div> <br> <button onclick="styleBox()">Style the Box</button> <script> function styleBox() { let box = document.getElementById("demo3"); box.style.color = "white"; box.style.backgroundColor = "green"; box.style.padding = "15px"; box.style.border = "2px solid darkgreen"; box.style.textAlign = "center"; } </script> </body> </html>

4. Overwriting Using cssText

The cssText property allows you to assign a string containing multiple CSS declarations to an element. Note that this will overwrite any existing inline styles on that element.

<!DOCTYPE html> <html> <body> <h3>The cssText Property</h3> <p id="demo4" style="color: red;">My red color will be overwritten.</p> <button onclick="applyCssText()">Apply cssText</button> <script> function applyCssText() { document.getElementById("demo4").style.cssText = "color: blue; font-size: 18px; font-weight: bold;"; } </script> </body> </html>

5. Hiding and Showing Elements

A very common use case for inline style manipulation is toggling the visibility of an element using the display property.

<!DOCTYPE html> <html> <body> <h3>Hide & Show Elements</h3> <div id="demo5" style="width: 100px; height: 100px; background-color: #ff7324;"></div> <br> <button onclick="hideDiv()">Hide</button> <button onclick="showDiv()">Show</button> <script> function hideDiv() { document.getElementById("demo5").style.display = "none"; } function showDiv() { document.getElementById("demo5").style.display = "block"; } </script> </body> </html>
Pro Tip: Directly manipulating the style property adds styles inline. Inline styles have the highest specificity and will override rules set in your external CSS files. For toggling many visual changes at once, using classList is generally preferred!
