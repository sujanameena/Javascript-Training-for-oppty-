HTML Input Types

The <input> element is the most important form element. The type attribute defines what kind of data the input field will accept.

1. Text Input (type="text")

This is the default type for an <input> element. It is used to create a single-line text field.

<label>Username:</label> <input type="text" placeholder="Enter your name" maxlength="20">
  • Purpose: Capturing basic information like names, usernames, or search queries.
  • placeholder: Displays a hint inside the box before the user types.
  • maxlength: Limits the number of characters allowed.

2. Password Input (type="password")

This type is specifically designed for sensitive data entry.

<label>Password:</label> <input type="password" id="pwd" name="password">
  • Purpose: It masks the characters (replacing them with dots or asterisks) so that bystanders cannot read the password.
  • Security Note: The data is still sent as plain text unless the website uses an encrypted connection (HTTPS).

3. Email Input (type="email")

This is a specialized text field introduced in HTML5 that includes built-in validation.

<label>Email Address:</label> <input type="email" name="user_email" required>
  • Purpose: Ensures the user enters a properly formatted email address (containing an "@" and a domain).
  • Mobile Benefit: Triggers a specialized keyboard on smartphones with @ and .com shortcuts.

4. Radio Button (type="radio")

Radio buttons allow a user to select exactly one option from a predefined set of choices.

<p>Select Gender:</p> <input type="radio" name="gender" value="male"> Male <input type="radio" name="gender" value="female"> Female
  • Grouping: To ensure only one can be picked, they must all share the same name attribute.
  • Purpose: Best for mutually exclusive answers like "Yes/No" or plan selections.

5. Number Input (type="number")

This type is used for fields that should only contain numeric values.

<label>Quantity (1-10):</label> <input type="number" min="1" max="10" step="1">
  • Attributes: min and max set the range; step sets the allowed intervals.
  • Benefit: Provides "spinners" (up/down arrows) to change values and prevents letter entry.
