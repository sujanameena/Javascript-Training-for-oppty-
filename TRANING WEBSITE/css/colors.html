CSS Colors

CSS allows several methods for specifying colors, including names, numerical values, and transparency options. Using different formats provides flexibility for branding and UI design.

Method Description Example
Color Names Use predefined names like red, blue, or green for simplicity. color: blue;
HEX Codes Define colors using six-digit hexadecimal codes (RRGGBB). #FF5733
RGB Define colors using Red, Green, and Blue intensities (0-255). rgb(255, 0, 0)
RGBA RGB with an Alpha channel for transparency (0.0 to 1.0). rgba(255, 0, 0, 0.5)
HSL Hue (0-360), Saturation (%), and Lightness (%). hsl(120, 100%, 50%)
HSLA HSL with an Alpha channel for transparency. hsla(120, 100%, 50%, 0.5)

Full Document Example: Color Formats

<!DOCTYPE html> <html> <head> <style> h2 { color: Tomato; } p.hex { color: #4CAF50; } p.rgb { color: rgb(0, 0, 255); } div.alpha { background-color: rgba(255, 0, 0, 0.3); padding: 10px; border: 1px solid red; } </style> </head> <body> <h2>CSS Color Methods</h2> <p class="hex">This text is green using HEX.</p> <p class="rgb">This text is blue using RGB.</p> <div class="alpha">This box has a transparent red background.</div> </body> </html>

2. CSS Units (Lengths)

Units are used to express lengths for properties like width, padding, and font-size. These are categorized into **Absolute** (fixed) and **Relative** (scalable) units.

A. Absolute Lengths

Unit Description
px Pixels (1px = 1/96th of 1 inch). The standard for screen display.
cm / mm Centimeters / Millimeters.
in Inches.

B. Relative Units

Unit Description
em Relative to the font-size of the element.
rem Relative to font-size of the root element (<html>).
vw / vh Relative to 1% of the viewport's width/height.
% Relative to the parent element.

Full Document Example: Relative Units

<!DOCTYPE html> <html> <head> <style> html { font-size: 16px; } .rem-text { font-size: 2rem; color: blue; } .vw-box { width: 50vw; background-color: #f1f1f1; border: 2px solid #ccc; padding: 10px; } </style> </head> <body> <p class="rem-text">This text is 32px (2rem x 16px root).</p> <div class="vw-box">This box is 50% of the viewport width.</div> </body> </html>
