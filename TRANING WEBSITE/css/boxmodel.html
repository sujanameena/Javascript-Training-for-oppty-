CSS Box Model

In CSS, the term "box model" is used when talking about web design and layout. The CSS box model is essentially a box that wraps around every HTML element. It consists of four distinct layers: **Content**, **Padding**, **Border**, and **Margin**.

1. Components of the Box Model

Each element in the box model is made up of distinct layers that define its size and spacing on a web page.

Standard CSS Box Model Layers.

A. Content Area

The core part that holds actual content (text, images, or child elements). It is bounded by the **content edge** and can be styled with height and width properties.

B. Padding Area

The space between the content and the border. It is transparent and contributes to the element's total dimensions.

C. Border Area

The outer "fencing" that wraps around the padding and content. Its thickness adds to the total height and width of the element.

D. Margin Area

The transparent space outside the border that separates the element from its neighbors. The box's total visible size stops at the border.

2. Box Sizing Property

The box-sizing property determines how the width and height of an element are calculated.

Property Type Calculation Behavior
Content-Box (Default) Width/Height apply only to the content. Padding and border are added **outside**.
Border-Box Width/Height include content, padding, and border. The content area shrinks to fit.

3. Box Model Examples

Example 1: Default Content-Box (Size Increase)

This box will be 250px wide total (200px content + 40px padding + 10px border).

<!DOCTYPE html> <html> <head> <style> div { width: 200px; padding: 20px; border: 5px solid black; box-sizing: content-box; background-color: lightgreen; } </style> </head> <body> <div>Total Width = 250px</div> </body> </html>

Example 2: Consistent Border-Box

The padding and border are included in the 200px width.

<!DOCTYPE html> <html> <head> <style> div { width: 200px; padding: 20px; border: 5px solid black; box-sizing: border-box; background-color: lightcoral; } </style> </head> <body> <div>Total Width = Stays 200px</div> </body> </html>

Example 3: Fixed Layout with Border-Box

Creating fixed-size elements without altering layout dimensions.

<!DOCTYPE html> <html> <head> <style> div { width: 300px; height: 100px; padding: 15px; border: 10px solid green; box-sizing: border-box; background-color: lightblue; } </style> </head> <body> <div>Fixed-size div with border-box.</div> </body> </html>

Example 4: Global Sizing Strategy

Apply border-box to all elements to simplify layout math.

<!DOCTYPE html> <html> <head> <style> * { box-sizing: border-box; } div { width: 100%; padding: 30px; border: 5px solid blue; background-color: lightyellow; } </style> </head> <body> <div>All elements on this page now include padding in their width.</div> </body> </html>

Example 5: Responsive Box

Ensuring padding does not cause overflow in responsive designs.

<!DOCTYPE html> <html> <head> <style> * { box-sizing: border-box; } .container { max-width: 100%; padding: 20px; border: 5px solid purple; background-color: lightgreen; } </style> </head> <body> <div class="container">Resizes according to screen width without breaking.</div> </body> </html>

4. Calculating Total Width and Height

Total Width Formula: width + left padding + right padding + left border + right border
Total Height Formula: height + top padding + bottom padding + top border + bottom border
