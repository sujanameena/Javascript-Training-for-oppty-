CSS Selectors

CSS selectors are used to "find" or select the HTML elements you want to style. They are a fundamental part of CSS, as they determine which rules apply to which parts of your document.

1. CSS Element Selector

The element selector selects HTML elements based on the element name. This applies the style to every instance of that element on the page.

Examples of Element Selection:
  • Styling all paragraphs (p) for consistent text flow.
  • Setting a global font for all headings (h1).
  • Removing default bullets from all unordered lists (ul).
  • Adding a consistent border to all images (img).
  • Setting a background color for all footer elements.
<!DOCTYPE html> <html> <head> <style> /* Selects all p elements */ p { text-align: center; color: red; } </style> </head> <body> <p>Every paragraph will be affected by the style.</p> <p>Me too!</p> <p>And me!</p> </body> </html>

2. CSS ID Selector

The ID selector uses the id attribute of an HTML element to select a specific, unique element. Since an ID must be unique within a page, it is used to style one single element. It is defined with a hash (#) character.

Common ID Use Cases:
  • Styling a unique main navigation bar (#navbar).
  • Targeting a specific unique header (#main-header).
  • Applying styles to a single contact form wrapper (#contact-form).
  • Highlighting a specific unique footer (#site-footer).
  • Positioning a unique sidebar (#left-sidebar).
<!DOCTYPE html> <html> <head> <style> /* Selects the element with id="para1" */ #para1 { text-align: center; color: blue; } </style> </head> <body> <p id="para1">Hello World!</p> <p>This paragraph is not affected by the style.</p> </body> </html>

3. CSS Class Selector

The class selector selects HTML elements with a specific class attribute. To select elements with a class, write a period (.) character followed by the class name. Unlike IDs, classes can be used on multiple elements.

Common Class Use Cases:
  • Creating a "center" class to center align any element.
  • Styling all warning messages with an "error" class.
  • Applying a "highlight" background to multiple text snippets.
  • Styling specific buttons with a "btn-primary" class.
  • Adding custom "card" styling to multiple product containers.
<!DOCTYPE html> <html> <head> <style> /* Selects all elements with class="center" */ .center { text-align: center; color: green; } </style> </head> <body> <h1 class="center">Center-aligned heading</h1> <p class="center">Center-aligned paragraph.</p> </body> </html>

4. Advanced: Class on Specific Elements

You can also specify that only certain HTML elements should be affected by a class.

<!DOCTYPE html> <html> <head> <style> /* Only p elements with class="large" will be styled */ p.large { font-size: 200%; color: orange; } </style> </head> <body> <h1 class="large">This heading is NOT orange</h1> <p class="large">This paragraph IS large and orange!</p> </body> </html>
