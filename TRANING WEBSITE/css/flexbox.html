CSS Flexbox (Flexible Box Layout)

CSS Flexbox is a one-dimensional layout model used for arranging items in rows OR columns. It makes it easy to design flexible and responsive layouts without using floats or complex positioning.

1. Flexbox Components and Axes

A Flexbox layout consists of a Flex Container (parent) and Flex Items (children).

The parent container establishes the alignment context for its children.

The Two Axes:
  • Main Axis: The primary direction items are laid out (defined by flex-direction).
  • Cross Axis: The axis perpendicular to the main axis.

2. Flex Container Properties (Parent)

These properties control the direction, alignment, and wrapping of the flex items.

Property Description & Values
flex-direction Sets the main axis: row (default), column, row-reverse, column-reverse.
flex-wrap Controls wrapping: nowrap (default), wrap, wrap-reverse.
justify-content Aligns items on **Main Axis**: flex-start, flex-end, center, space-between, space-around.
align-items Aligns items on **Cross Axis**: stretch (default), center, flex-start, flex-end, baseline.
align-content Aligns multiple flex lines (only works if flex-wrap is used).

3. Flex Item Properties (Children)

These properties allow individual children to behave differently within the container.

Property Description
order Specifies the visual order of a flex item (default is 0).
flex-grow Determines how much an item will grow relative to the others (default is 0).
flex-shrink Determines how much an item will shrink relative to the others.
flex-basis Defines the initial size of an item before space is distributed.
align-self Allows the default alignment to be overridden for individual items.

4. Flexbox Layout Examples

Example 1: Flexbox Container Basics

A simple horizontal container with three items.

<!DOCTYPE html> <html> <head> <style> .container { display: flex; background-color: DodgerBlue; } .container > div { background-color: #f1f1f1; margin: 10px; padding: 20px; font-size: 30px; } </style> </head> <body> <div class="container"> <div>1</div> <div>2</div> <div>3</div> </div> </body> </html>

Example 2: Horizontal Alignment (Justify Content)

Aligning items to the center of the main axis.

<!DOCTYPE html> <html> <head> <style> .container { display: flex; justify-content: center; background-color: lightgrey; } .container > div { background-color: DodgerBlue; color: white; width: 100px; margin: 10px; text-align: center; line-height: 75px; font-size: 30px; } </style> </head> <body> <div class="container"> <div>1</div> <div>2</div> <div>3</div> </div> </body> </html>

Example 3: Vertical Alignment (Align Items)

Centering items along the cross axis (vertically).

<!DOCTYPE html> <html> <head> <style> .container { display: flex; height: 200px; align-items: center; background-color: #f1f1f1; border: 1px solid #ccc; } .container > div { background-color: #04AA6D; color: white; width: 100px; margin: 10px; text-align: center; line-height: 50px; } </style> </head> <body> <div class="container"> <div>1</div> <div>2</div> <div>3</div> </div> </body> </html>

Example 4: Flex Wrap

Items will wrap onto multiple lines if there is not enough space.

<!DOCTYPE html> <html> <head> <style> .container { display: flex; flex-wrap: wrap; background-color: DodgerBlue; } .container > div { background-color: #f1f1f1; width: 100px; margin: 10px; text-align: center; line-height: 75px; font-size: 30px; } </style> </head> <body> <div class="container"> <div>1</div><div>2</div><div>3</div><div>4</div> <div>5</div><div>6</div><div>7</div><div>8</div> </div> </body> </html>

Example 5: Holy Grail Layout

Using flex-direction: column for the main page and flex: 1 to fill the middle area.

<!DOCTYPE html> <html> <head> <style> body { display: flex; flex-direction: column; min-height: 100vh; margin: 0; } header, footer { background: #333; color: white; padding: 20px; text-align: center; } .main-content { display: flex; flex: 1; } article { flex: 3; background: #f4f4f4; padding: 20px; } nav { flex: 1; background: #ddd; order: -1; padding: 20px; } /* Navigation on left */ aside { flex: 1; background: #ddd; padding: 20px; } /* Sidebar on right */ </style> </head> <body> <header>Header Section</header> <div class="main-content"> <article>Main Article Content (flex: 3)</article> <nav>Left Sidebar (flex: 1)</nav> <aside>Right Sidebar (flex: 1)</aside> </div> <footer>Footer Section</footer> </body> </html>
