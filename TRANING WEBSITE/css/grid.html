CSS Grid Layout

The CSS Grid Layout Module offers a powerful two-dimensional layout system, utilizing both rows and columns. It allows developers to create complex web structures easily without relying on floats or positioning.

1. Grid Architecture

A Grid layout consists of a Grid Container (parent) and Grid Items (children). This relationship creates a coordinate system for precise placement.

The relationship between the Grid Container and Grid Items.

2. Grid Sizing Units

Grid tracks are the spaces between grid lines. You define them using specific units and functions to control flexibility.

Sizing Unit Behavior & Use Case
fr (Fractional) Allocates a fraction of the available free space in the container.
auto Sizes the track based on the content length of items.
repeat() Shorthand for defining multiple tracks of the same size.
minmax() Defines a size range with a minimum and maximum limit.

3. Important Grid Properties

A. Container Properties (Parent)

Properties like grid-template-columns and grid-template-rows define the structure of the grid.

gap Specifies the spacing (gutters) between grid cells.
place-items Aligns items along both axes (horizontal and vertical) simultaneously.

B. Item Properties (Children)

Properties used to control the position and spanning of individual elements within the grid.

grid-column / grid-row Determines an item's start and end position and span.
grid-area Assigns a specific name to an element for template-based placement.

4. CSS Grid Layout Examples

Example 1: Basic Three-Column Layout

This creates a simple 3-column grid with equal widths using 1fr.

<!DOCTYPE html> <html> <head> <style> .container { display: grid; grid-template-columns: 1fr 1fr 1fr; gap: 15px; background-color: #2196F3; padding: 15px; } .grid-item { background-color: #f1f1f1; border: 1px solid black; padding: 20px; font-size: 24px; text-align: center; } </style> </head> <body> <div class="container"> <div class="grid-item">1</div> <div class="grid-item">2</div> <div class="grid-item">3</div> <div class="grid-item">4</div> <div class="grid-item">5</div> <div class="grid-item">6</div> </div> </body> </html>

Example 2: Spanning Columns and Rows

The first item spans across two columns and two rows to create a featured block.

<!DOCTYPE html> <html> <head> <style> .container { display: grid; grid-template-columns: auto auto auto auto; gap: 10px; background-color: #e7e9eb; padding: 10px; } .item1 { grid-column: span 2; grid-row: span 2; background-color: #04AA6D !important; color: white; } .grid-item { background-color: white; padding: 20px; text-align: center; border: 1px solid #ccc; } </style> </head> <body> <div class="container"> <div class="grid-item item1">Large Item</div> <div class="grid-item">2</div> <div class="grid-item">3</div> <div class="grid-item">4</div> <div class="grid-item">5</div> </div> </body> </html>

Example 3: Dashboard Grid Areas

Structuring a full page layout using semantic named grid areas.

<!DOCTYPE html> <html> <head> <style> .container { display: grid; grid-template-areas: 'header header header' 'sidebar main main' 'footer footer footer'; gap: 10px; } .header { grid-area: header; background: coral; } .sidebar { grid-area: sidebar; background: lightblue; } .main { grid-area: main; background: #ddd; min-height: 150px; } .footer { grid-area: footer; background: lightgreen; } .container > div { padding: 20px; text-align: center; } </style> </head> <body> <div class="container"> <div class="header">Header</div> <div class="sidebar">Sidebar</div> <div class="main">Main Content</div> <div class="footer">Footer</div> </div> </body> </html>

Example 4: minmax() for Responsiveness

The column will never be smaller than 100px or larger than 1fr.

<!DOCTYPE html> <html> <head> <style> .container { display: grid; grid-template-columns: minmax(100px, 1fr) 2fr; gap: 10px; } .grid-item { background: #04AA6D; color: white; padding: 20px; } </style> </head> <body> <div class="container"> <div class="grid-item">Min 100px / Max 1fr</div> <div class="grid-item">Flexible 2fr</div> </div> </body> </html>

Example 5: Precise Content Alignment

Centering the entire grid block inside a container using place-content.

<!DOCTYPE html> <html> <head> <style> .container { display: grid; grid-template-columns: repeat(2, 100px); height: 300px; place-content: center; background-color: #f1f1f1; gap: 10px; } .grid-item { background: tomato; color: white; padding: 20px; text-align: center; } </style> </head> <body> <div class="container"> <div class="grid-item">Cell A</div> <div class="grid-item">Cell B</div> <div class="grid-item">Cell C</div> <div class="grid-item">Cell D</div> </div> </body> </html>

5. Layout Logic

Grid Tip: Unlike Flexbox, which is primarily one-dimensional (row OR column), Grid allows you to align items against two axes simultaneously, making it the ideal choice for layout-heavy designs.
