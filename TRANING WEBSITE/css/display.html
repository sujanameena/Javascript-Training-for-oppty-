CSS Display & Visibility: Ultimate Master Class

The display property is the most important CSS property for controlling layout. It determines how an element is rendered in the document flow, while the visibility property controls whether an element is seen or hidden without changing the document structure.

1. The Display Property Matrix

Every HTML element has a default display value depending on what type of element it is. Most elements are usually block or inline.

Value Description
inlineDisplays an element as an inline element (like <span>). Height and width properties have no effect. Default for text.
blockDisplays an element as a block element (like <p>). It starts on a new line and takes up the full width available.
inline-blockFormats as an inline element, but allows you to apply height and width values.
noneThe element is completely removed from the layout. It takes up no space.
flexDisplays an element as a block-level flex container.
gridDisplays an element as a block-level grid container.
contentsMakes the container disappear, making child elements direct children of the next level up.
list-itemLet the element behave like a <li> element.
tableLet the element behave like a <table> element.
initial/inheritSets the property to its default value or inherits from the parent.

2. Interactive Examples: Core Display Values

The difference between inline, block, and inline-block is the foundation of CSS layouts.

Example 1: Block vs. Inline vs. Inline-Block

<!DOCTYPE html> <html> <head> <style> .box { background-color: #ff6b6b; color: white; padding: 10px; border: 1px solid black; } .block-box { display: block; margin-bottom: 5px; } .inline-box { display: inline; } .inline-block-box { display: inline-block; width: 150px; height: 50px; margin: 5px; } </style> </head> <body> <h4>Display: Block (Full width, new line)</h4> <div class="box block-box">Box 1</div> <div class="box block-box">Box 2</div> <h4>Display: Inline (Width only as wide as content)</h4> <div class="box inline-box">Inline Box 1</div> <div class="box inline-box">Inline Box 2</div> <h4>Display: Inline-Block (Inline flow + Custom Size)</h4> <div class="box inline-block-box">150x50px Box</div> <div class="box inline-block-box">150x50px Box</div> </body> </html>

3. Display: None vs. Visibility: Hidden

A common point of confusion is the difference between removing an element and simply hiding it.

Property Space in Layout Interaction
display: none; Space is collapsed (removed). Cannot be clicked/focused.
visibility: hidden; Space is preserved (invisible gap). Cannot be clicked/focused.

Example 2: Hiding Elements

<!DOCTYPE html> <html> <head> <style> .box { width: 100px; height: 50px; margin: 10px; background: #667eea; color: white; display: inline-block; } .hidden-none { display: none; } .hidden-vis { visibility: hidden; } </style> </head> <body> <div class="box">Box 1</div> <div class="box hidden-none">Box 2 (none)</div> <div class="box">Box 3</div> <p>Above: Box 2 uses <b>display: none</b>. Box 3 moves to take its spot.</p> <div class="box">Box A</div> <div class="box hidden-vis">Box B (hidden)</div> <div class="box">Box C</div> <p>Above: Box B uses <b>visibility: hidden</b>. The space remains empty.</p> </body> </html>

4. Advanced Display: Flex & Grid

Modern layouts use flex for one-dimensional layouts (rows or columns) and grid for two-dimensional layouts.

Example 3: Flexbox Basics

<!DOCTYPE html> <html> <head> <style> .flex-container { display: flex; justify-content: space-around; background-color: #e9ecef; padding: 10px; } .flex-item { background: #4ecdc4; padding: 20px; color: white; font-weight: bold; } </style> </head> <body> <div class="flex-container"> <div class="flex-item">1</div> <div class="flex-item">2</div> <div class="flex-item">3</div> </div> </body> </html>

5. Pro-Tips & Best Practices

1. Accessibility: Elements with display: none are hidden from screen readers. Use CSS techniques like "sr-only" if you want text hidden visually but available to assistive technology.

2. Centering: display: block elements can be centered horizontally using margin: auto if they have a set width.

3. Hover Effects: Use visibility or opacity instead of display: none if you want to animate an element appearing, as display cannot be transitioned.
