CSS Media Queries: Ultimate Master Class

Media Queries are a powerful feature in CSS that allow you to apply different styles based on device characteristics like screen width, height, orientation, and resolution. They are essential for creating responsive web designs that adapt to any device.

1. The Anatomy of a Media Query

In CSS, a media query is an @media rule that acts like an "If-Then" statement: If the device meets these conditions, then apply these styles.

@media media-type and (media-feature) {
  /* CSS rules go here */
}
Component Description Examples
Media Types Defines the category of the device. screen, print, all
Media Features Specific characteristics to test. max-width, orientation, min-resolution
Logical Operators Combine or negate conditions. and, not, only, , (or)

2. Standard Device Breakpoints

Breakpoints are the "sweet spots" where your layout adjusts to provide the best user experience.

Device Type Breakpoint CSS Rule Example
Extra Small (Phones) up to 600px @media (max-width: 600px)
Medium (Tablets) 768px and up @media (min-width: 768px)
Large (Desktops) 992px and up @media (min-width: 992px)

3. Mobile-First vs. Desktop-First

Mobile-First (Recommended)

Start with styles for small screens and add complexity using min-width.

.nav { display: none; }
@media (min-width: 768px) { .nav { display: flex; } }

Desktop-First

Start with desktop styles and simplify for small screens using max-width.

.nav { display: flex; }
@media (max-width: 768px) { .nav { display: none; } }

4. Interactive Examples

Example 1: Responsive Layout & Orientation

This box reacts to both screen width and whether your device is tilted (Portrait vs. Landscape).

<!DOCTYPE html> <html> <head> <style> .box { padding: 20px; text-align: center; color: white; transition: 0.3s; } /* Default Mobile Styles */ .box { background-color: #ff9800; } .box::before { content: "Mobile / Portrait View"; } /* Tablet Landscape */ @media screen and (min-width: 768px) and (orientation: landscape) { .box { background-color: #2196F3; } .box::before { content: "Tablet Landscape View"; } } /* Large Desktop */ @media screen and (min-width: 1024px) { .box { background-color: #4CAF50; font-size: 24px; } .box::before { content: "Desktop Master View"; } } </style> </head> <body> <div class="box"></div> </body> </html>

Example 2: Responsive Grid & Typography

A 3-column grid that stacks on mobile, and text that grows with the screen.

<!DOCTYPE html> <html> <head> <style> .grid { display: flex; flex-wrap: wrap; gap: 10px; } .item { flex: 1; min-width: 100%; padding: 20px; background: #eee; text-align: center; } p { font-size: 14px; } @media (min-width: 768px) { .item { min-width: 30%; } /* 3 Columns on Tablet/Desktop */ p { font-size: 20px; } /* Larger Text */ } </style> </head> <body> <p>Responsive Typography Example</p> <div class="grid"> <div class="item">Card 1</div> <div class="item">Card 2</div> <div class="item">Card 3</div> </div> </body> </html>

5. Advanced Media Features

Feature Usage
print Hides sidebars or changes fonts for paper printing.
min-resolution Targets Retina/High-DPI displays to serve sharper images.
prefers-color-scheme Detects if the user has Dark Mode enabled in their OS.

6. Pro-Tips & Summary

  • The Viewport Meta Tag: Always include <meta name="viewport" content="width=device-width, initial-scale=1.0"> in your HTML head.
  • Range Syntax: Modern browsers now support @media (width <= 600px) as a cleaner alternative to max-width.
  • Accessibility: Respect prefers-reduced-motion to disable heavy animations for sensitive users.
