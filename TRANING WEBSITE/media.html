HTML Media Elements

HTML5 allows you to embed images, audio, and video directly into your web pages to create a rich multimedia experience.

1. The HTML <img> Tag

The <img> tag is used to embed images. It is an empty tag, meaning it contains attributes only and does not have a closing tag.

<img src="https://picsum.photos/500/300" alt="A beautiful mountain landscape" width="500" height="300">
  • src (source): Specifies the path (URL) to the image.
  • alt (alternate text): Critical for SEO and Accessibility. It shows if the image fails to load.
  • Sizing: Use width and height to control dimensions directly.

2. The <audio> Element

This element allows you to play audio files directly in the browser without external plugins.

<audio controls> <source src="https://upload.wikimedia.org/wikipedia/commons/c/c8/Example.ogg" type="audio/ogg"> <source src="https://www.w3schools.com/html/horse.mp3" type="audio/mpeg"> Your browser does not support the audio element. </audio>
  • The controls Attribute: Essential to make play/pause/volume buttons visible.
  • The <source> Tag: Used to provide multiple formats for cross-browser compatibility.

3. The <video> Element

Similar to audio, the <video> tag is used to embed video clips.

<video width="400" height="240" controls poster="https://peach.blender.org/wp-content/uploads/title_an_01.jpg"> <source src="https://www.w3schools.com/html/mov_bbb.mp4" type="video/mp4"> Your browser does not support the video tag. </video>
  • autoplay: Starts the video immediately (usually requires muted).
  • poster: Displays an image until the video is played.

4. Supporting Multiple Formats

To ensure media plays for everyone, provide alternatives within the tags:

  • Video: MP4 (widest support), WebM, and Ogg.
  • Audio: MP3 (standard), WAV, and Ogg.
  • Fallback Text: Text placed inside the tags appears if the browser is too old to support HTML5 media.

5. Using Images as Links

Turn an image into a clickable link by wrapping the <img> tag inside an <a> (anchor) tag.

<img src="https://images.unsplash.com/photo-1464822759023-fed622ff2c3b?auto=format&fit= crop&w=500&h=300" alt="Mountain Landscape" width="500" height="300">
