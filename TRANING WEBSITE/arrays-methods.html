JavaScript Array Iteration Methods

Array iteration methods are used to operate on every element of an array. These modern ES6 methods provide a cleaner way to transform and reduce data.

1. The map() Method

The map() method creates a new array by performing a function on each array element.

const numbers = [45, 4, 9, 16, 25]; const numbers2 = numbers.map(myFunction); function myFunction(value) { return value * 2; } console.log(numbers2); // [90, 8, 18, 32, 50]

2. The filter() Method

The filter() method creates a new array with array elements that pass a test.

const numbers = [45, 4, 9, 16, 25]; const over18 = numbers.filter(myFunction); function myFunction(value) { return value > 18; } console.log(over18); // [45, 25]

3. The reduce() Method

The reduce() method runs a function on each array element to produce a single value.

const numbers = [45, 4, 9, 16, 25]; let sum = numbers.reduce(myFunction); function myFunction(total, value) { return total + value; } console.log(sum); // 99
Oppty Academy Pro Tip: These methods do not change the original array. This concept is known as immutability, a core principle of modern functional programming!
