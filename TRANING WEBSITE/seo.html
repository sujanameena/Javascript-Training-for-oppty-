HTML Meta Tags & SEO

Meta tags provide metadata about the HTML document. They are not displayed on the page but are used by browsers and search engines.

1. Character Encoding Tag (<meta charset="UTF-8">)

This tag tells the web browser how to display the characters and symbols on your webpage.

<head> <meta charset="UTF-8"> </head>
  • Purpose: Ensures all text, including special characters and emojis, renders correctly.
  • Benefit: Prevents "broken" symbols, ensuring a professional user experience.

2. Description Meta Tag

The Description tag provides a short summary (snippet) of what the webpage is about.

<meta name="description" content="Free Web tutorials for HTML, CSS, and JavaScript.">
  • SEO Benefit: A compelling description improves your Click-Through Rate (CTR) in search results.
  • Best Practice: Keep length between 150–160 characters.

3. Viewport Meta Tag

This tag is the foundation of Responsive Web Design, crucial for mobile devices.

<meta name="viewport" content="width=device-width, initial-scale=1.0">
  • Purpose: Controls page scaling on different device widths.
  • SEO Benefit: Essential for Google’s mobile-first indexing ranking.

4. Robots Meta Tag

The Robots tag acts as a "signpost" for search engine crawlers (bots) visiting your site.

<meta name="robots" content="index, follow">
  • index/noindex: Tells the bot whether to include the page in search results.
  • follow/nofollow: Instructs the bot to follow (or ignore) links on the page.

5. Author and Keywords Tags

These tags provide additional context, though they have less direct impact on modern SEO rankings.

<meta name="author" content="John Doe"> <meta name="keywords" content="HTML, SEO, Meta Tags">
