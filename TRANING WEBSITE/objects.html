JavaScript Objects & JSON

In JavaScript, an Object is a non-primitive data type that acts as a standalone entity with properties and type. It is an unordered collection of related data in the form of key:value pairs.

1. Creating Objects (Object Literals)

Creating an object using the literal {} is the simplest way. Properties consist of a name (key) and a value.

// Creating a JavaScript Object const person = { firstName: "John", lastName: "Doe", age: 30, city: "New York" }; console.log("Name: " + person.firstName + " " + person.lastName); console.log("Age: " + person.age);

2. Accessing Properties (Dot vs Bracket)

Use Dot Notation (obj.prop) for standard keys, and Bracket Notation (obj["prop"]) for keys with spaces or dynamic names.

const car = { brand: "BMW", "fuel type": "Petrol" }; // Dot Notation console.log(car.brand); // Bracket Notation (Required for keys with spaces) console.log(car["fuel type"]);

3. Object Methods and 'this'

A method is a function stored as a property. The this keyword refers to the object owning the method.

const user = { userName: "Sandeep", greet: function() { return "Hello, my name is " + this.userName; } }; console.log(user.greet());

4. Nested Objects and Arrays

Objects can store other objects and arrays, allowing you to model complex real-world data structures.

const student = { id: 101, details: { firstName: "Amit", lastName: "Sharma" }, subjects: ["Math", "Physics", "Computer"] }; console.log(student.details.firstName); console.log(student.subjects[0]);

Working with JSON

JSON (JavaScript Object Notation) is a text format used for transporting data. Since servers only send/receive strings, we use JSON methods to convert our data.

JSON.stringify() - Object to String

Converts a JavaScript object into a JSON string to be sent to a server.

const user = { name: "Oppty", active: true, id: 12 }; // Converting Object to JSON String const jsonString = JSON.stringify(user); console.log(jsonString); console.log(typeof jsonString); // "string"

JSON.parse() - String to Object

Converts a JSON string received from a server back into a JavaScript object.

const text = '{"name":"Sandeep", "age":30, "city":"Delhi"}'; // Converting JSON String to Object const obj = JSON.parse(text); console.log(obj.name); console.log(obj.city);
Oppty Academy Pro Tip: In JSON, all keys must be wrapped in double quotes ("key"), and you cannot store functions or methods inside JSON data!
