JavaScript classList Manipulation

You can add, remove, or toggle classes on an element using JavaScript's classList property. This is highly helpful for applying styles or applying CSS animations dynamically without manipulating raw strings.

1. Adding a Class (classList.add)

The classList.add() method adds one or more classes to an element. If the specified class already exists, it simply ignores it.

<!DOCTYPE html> <html> <head> <style> .highlight { color: red; font-weight: bold; background: yellow; padding: 5px; } </style> </head> <body> <h3>Add a CSS Class</h3> <div id="example1">This is a normal text element.</div> <br> <button onclick="addClass()">Add 'highlight' Class</button> <script> function addClass() { document.getElementById("example1").classList.add("highlight"); } </script> </body> </html>

2. Removing a Class (classList.remove)

The classList.remove() method removes one or more classes from an element. If the class doesn't exist, it won't throw an error.

<!DOCTYPE html> <html> <head> <style> .bold-text { font-weight: bold; font-size: 20px; color: blue; } </style> </head> <body> <h3>Remove a CSS Class</h3> <div id="example2" class="bold-text">This text has the "bold-text" class applied.</div> <br> <button onclick="removeClass()">Remove Class</button> <script> function removeClass() { document.getElementById("example2").classList.remove("bold-text"); } </script> </body> </html>

3. Toggling a Class (classList.toggle)

The classList.toggle() method toggles a class on an element. If the class is not present, it adds it. If it is already present, it removes it.

<!DOCTYPE html> <html> <head> <style> .active { border-left: 5px solid green; padding-left: 10px; color: green; } </style> </head> <body> <h3>Toggle a CSS Class</h3> <div id="example3">Click the button to toggle my active state.</div> <br> <button onclick="toggleClass()">Toggle 'active' Class</button> <script> function toggleClass() { document.getElementById("example3").classList.toggle("active"); } </script> </body> </html>

4. Checking for a Class (classList.contains)

The classList.contains() method returns a boolean value indicating whether an element has the specified class. This is excellent for conditional logic.

<!DOCTYPE html> <html> <body> <h3>Check if Class Exists</h3> <div id="example4" class="hidden-module">I am an element with a class.</div> <p id="output"></p> <button onclick="checkClass()">Check Contains</button> <script> function checkClass() { const el = document.getElementById("example4"); const hasClass = el.classList.contains("hidden-module"); document.getElementById("output").innerText = "Does it have 'hidden-module'? " + hasClass; } </script> </body> </html>

5. Replacing a Class (classList.replace)

The classList.replace() method replaces an existing class with a new class. It returns true if the class was successfully replaced.

<!DOCTYPE html> <html> <head> <style> .theme-light { background: white; color: black; border: 1px solid gray; padding: 10px; } .theme-dark { background: black; color: white; border: 1px solid gray; padding: 10px; } </style> </head> <body> <h3>Replace a CSS Class</h3> <div id="example5" class="theme-light">This is a themed box.</div> <br> <button onclick="replaceClass()">Switch to Dark Theme</button> <script> function replaceClass() { document.getElementById("example5").classList.replace("theme-light", "theme-dark"); } </script> </body> </html>
Pro Tip: Using classList is much safer and easier than manipulating the raw className string, because it automatically handles spaces and prevents accidentally overwriting other classes!
