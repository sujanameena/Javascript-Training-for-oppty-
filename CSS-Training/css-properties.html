CSS=Cascading style sheet Example: ------------------------------------------------------------------------------------------------------ Selectors in CSS ------------------------------------------------------------------------------------------------------ 1. Universal Selector: Applies styles to all elements. * { margin: 0; padding: 0; } 2. Type Selector: Targets specific HTML elements. h1 { font-family: Arial, sans-serif; } 3. Class Selector: Styles elements with a specific class attribute. .box { border: 1px solid black; padding: 10px; } 4. ID Selector: Targets a single element with a specific ID. #header { background-color: lightgray; } ------------------------------------------------------------------------------------------------------ Grouping and Nesting of CSS Selectors ------------------------------------------------------------------------------------------------------ 1. Single-line text with text-overflow: ellipsis ========================================================================== Font-related properties ========================================================================== font-family – which font(s) to use font-size – size of the text font-weight – thickness (normal, bold, 100–900) font-style – normal, italic, oblique font-variant – small-caps,all-small-caps,numeric etc. line-height – space between lines(give pixels) font – shorthand for font-style, weight, size, line-height, family (More advanced) font-stretch, font-kerning, font-feature-settings, font-variant-ligatures, font-variant-caps ========================================================================== Text-related properties ========================================================================== color – text color text-align – left, right, center, justify text-decoration – underline, line-through, overline text-transform – uppercase, lowercase, capitalize text-indent – indent first line of a paragraph letter-spacing – space between letters word-spacing – space between words line-height – (also here) line spacing white-space – how spaces and line breaks behave text-overflow – what to do with overflowed text (e.g., ellipsis) text-shadow – shadow behind text direction – text direction (ltr, rtl) unicode-bidi – advanced control of bidirectional text vertical-align – vertical alignment of inline/inline-block elements ========================================================================== Shadows ========================================================================== It supports two kinds of shadows per shadow: Outer shadow (default) Inner shadow (when you add inset) box-shadow: 4px 4px 8px rgba(0, 0, 0, 0.3); box-shadow: 0 0 0 5px rgba(0,128,255,0.5); 4px (first value) offset-x → moves the shadow 4px to the right. (Negative value would move it to the left.) 4px (second value) offset-y → moves the shadow 4px down. (Negative value would move it up.) 8px (third value) blur-radius → how soft the shadow is. Bigger value = more blurred and spread out; 0 = sharp edge. 5px (fourth value – spread-radius) Shadow grows 5px outward in all directions, like an extra border.
