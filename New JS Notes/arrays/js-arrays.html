A JavaScript array is an ordered list-like object used to store multiple values in a single variable, where each value has a numeric index starting from 0. It can hold any data types (numbers, strings, objects, even other arrays) together. Arrays are dynamic: you can add, remove, and change elements at any time. Example:1 1. Create Array using Literal * The array literal is the most common and simple way to create an array. * You use square brackets [] and list the elements separated by commas Syntax: let arrayName = [value1, value2, value3]; Example:1 – simple string array Example:2 – mixed types in one array 2. Create using new Keyword (Constructor) Another way to create an array is using the Array constructor with the new keyword. Less common than literals, but good to know.
