CSS file (style.css): h1 { color: purple; text-align: center; } p { color: teal; font-size: 18px; } “Cascading” means when multiple styles apply, the browser decides which one wins based on order, specificity, and importance. Basic rule syntax: ------------------ selector { property: value; } property+value=declaration block p { color: blue; font-size: 16px; } Here are the main advantages of CSS: ------------------------------------ * Separates content and design, making HTML cleaner and easier to read. * Reusability: one CSS file can style many HTML pages. * Easier maintenance: change style in one place, update everywhere. * Consistent look and feel across the whole website. * Better control over layout (flexbox, grid, positioning, responsive design). * Smaller HTML files and faster page loading (less repeated inline styling). * Improved accessibility and SEO when used with semantic HTML. -----! SEO stands for Search Engine Optimization. It is the process of improving a website so that it appears higher in search engine results (Google, Bing, etc.). Goal: get more visitors from search engines without paying for ads. It involves: Using good, relevant content and keywords Clean, semantic HTML structure Fast loading pages and mobile-friendly design Proper titles, headings, meta tags, alt text for images, etc.
