This text is aligned to the start edge (left for LTR, right for RTL).

This text is aligned to the end edge (right for LTR, left for RTL).

************************** font-variant css property ************************** The CSS font-variant property controls the use of alternate glyphs (letter shapes) for text, allowing you to enable small caps, old-style numbers, and other typographic features. Common values: normal: Default, no special variants. small-caps: Displays lowercase letters as smaller uppercase letters. all-small-caps: All letters (including uppercase) are shown as small caps (supported in modern browsers). inherit: Inherits the value from its parent. Usage: Use small-caps for headings, acronyms, or stylistic emphasis. all-small-caps is more consistent for text that mixes upper and lower case. Example:
