***************** HTML Attributes ***************** Definition: ----------- TML attributes are special properties added to an HTML element’s opening tag to provide extra information or control its behavior. Syntax: Content Common attributes: id: unique name for one element. class: reusable group name for CSS/JS. style: inline CSS. title: tooltip text. href: link destination. src: source file path for media/script. alt: fallback text for images. name: form field name sent to server. value: initial value of an input. placeholder: hint text in input. required: input must be filled. disabled: control cannot be used. readonly: value visible but not editable. checked: default selected checkbox/radio. target: where link opens. 1.id: unique name for one element. ---------------------------------- Definition: The id attribute gives a unique name to one HTML element on a page. No two elements should use the same id value. Example:1

Welcome

2.class: reusable group name for CSS/JS. ---------------------------------------- The class attribute is used to assign one or more reusable names to an HTML element, so you can apply the same CSS style or JavaScript behavior to many elements. Example:1

This is a note.

This is another note.

This is normal text.
