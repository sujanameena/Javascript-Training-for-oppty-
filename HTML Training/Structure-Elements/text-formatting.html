HTML Text Formatting --------------------- HTML text formatting means using special tags to change how text looks or behaves, without changing the actual content. It includes tags like: * b, strong – bold/important text * i, em – italic/emphasized text * u – underlined text * mark – highlighted text(with background) * small – smaller text * sub, sup – subscript and superscript * del, ins – deleted(Strikes through text) and highlights inserted text Example: Categories of HTML Text Formatting ---------------------------------- In HTML logical tags describe the meaning (semantics) of text, while physical tags describe only the appearance (how it looks). Logical (semantic) tags - meaning/importance * strong - strong importance * em - emphasis * cite, abbr, code, etc. Physical (presentational) tags - visual style only * b - bold * i - italic * u - underline * small, sub, sup, etc. Example: Combining Logical and Physical Tags: ------------------------------------ Example:
